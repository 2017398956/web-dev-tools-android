# Web Developer Tools Android

An example Android app with MVVM, clean architecture, modularization, repository pattern


## Tech Stack
```
- Kotlin
- MVVM
- Modularization
- Repository Pattern
- Coroutines
- Koin
- Architecture Components
- Navigation Component
- LiveData
- Paging
- Kotlin DSL
- OkHttp3
```
