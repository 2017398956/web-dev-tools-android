# Web Developer Tools Android

A lightweight HTTP Rest API, Websocket Test Client


## Tech Stack
```
- Kotlin
- MVVM
- Modularization
- Repository Pattern
- Coroutines
- Koin
- Architecture Components
- Navigation Component
- LiveData
- Paging
- Kotlin DSL
- OkHttp3
```
